## HOW TO BUILD A PROMPT (step by step)

1. **Identify use case** → which of the 5 characters? Or which app feature to promote?
2. **Pick template** → 15s (paid), 27s (organic reel), 45s (story/YouTube)
3. **Set the hook** → first line is everything. Steal from the character's pain point.
4. **Map scenes to templates** → assign correct Screen IDs per scene
5. **Write exact VO** → slow, deliberate, pauses marked with [pause]
6. **Write exact text overlays** → gold title first, white subtitle beneath
7. **Apply audio notes** → singing bowl open, 528Hz bed, final bowl close
8. **Add brand footer** → always include @meditate_with_abhi and free download CTA
9. **Output as clean code block** → ready to paste into HeyGen Video Agent
