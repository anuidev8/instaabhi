# The School of Breath — Visual Rules

## Layout
- With character (default): left 40-50% character, right 50-60% text
- Without character: text-first composition with one support visual
- Keep safe-zone spacing on all sides for mobile readability

## Text Hierarchy
- Tier 1 (small): context/setup line
- Tier 2 (largest): main hook (2-5 words)
- Tier 3 (optional strip): urgency/curiosity CTA

## Character Rules
- Use only approved Abhi references
- Preserve identity and teacher presence
- Avoid fantasy/deity styling

## No-Character Rules
- Disable people completely
- Use one symbolic support visual only
- Favor readability over visual complexity

## Contrast
- Main hook must stay legible at tiny phone preview size
- Simplify background whenever readability drops
