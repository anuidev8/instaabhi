# The School of Breath — Brand Core

## Purpose
The School of Breath thumbnails must drive fast mobile clicks with clear, practical, body-focused messaging.

## Personality
- Practical
- Immediate
- High-impact
- Modern
- Results-driven

## Core Promise
Each thumbnail communicates in one glance:
- problem
- method
- result

## Not This
- Mystical devotional style
- Soft abstract wellness aesthetic
- Cluttered scenes with many competing elements
- Temple/fantasy religious visuals
